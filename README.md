# 🤖 Guided Chatbot using IBM watsonx

## 📌 Overview
This project is a **guided chatbot system** developed during the **IBM SkillsBuild Internship (Edunet Foundation & AICTE)**.

Unlike generative AI chatbots, this system works using **predefined options and responses**, where users interact by selecting choices, and the system provides relevant answers.

---

## 🧠 Problem Statement
Many users need quick and accurate information without the complexity of typing queries or receiving unpredictable AI-generated responses.

This project aims to build a **structured chatbot system** that:
- Provides clear options to users
- Reduces ambiguity in responses
- Ensures consistent and accurate outputs

---

## 💡 Solution
The chatbot is designed as a **decision-based system**:

- Users are presented with predefined options
- Each option is mapped to a specific response
- Based on user selection, the system returns the corresponding answer

This ensures:
- ✅ Fast interaction  
- ✅ No incorrect AI-generated responses  
- ✅ Controlled and reliable outputs

## 🔄 How It Works

1. User opens the chatbot
2. System displays a list of options
3. User selects an option
4. System maps the selection to a predefined response
5. Answer is displayed to the user
